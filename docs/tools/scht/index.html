SAROO Cheat Builder

Build SAROO `.scht` files from user-provided cheat codes. The tool can convert manual AR/GameShark-style code input, RetroArch cheat files, and user-supplied Pseudo Saturn Kai source files. `.scht` cheat files are currently supported only by MarTinX SAROO firmware.
